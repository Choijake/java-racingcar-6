## 클래스

### User

- [ ] 레이싱에 참여할 자동차의 이름을 알고 있다.
- [ ] 레이싱에 참여할 이름을 가진 자동차를 만들게 할 수 있다.
- [ ] 시도할 횟수를 알고 있다.
- [ ] 횟수 만큼 시도 하도록 할 수 있다.



### CarFactory

- [ ] 레이싱에 참여할 자동차의 이름을 알고 있다.
- [ ] 레이싱에 참여할 이름을 가진 자동차를 만들 수 있다.


### Car

- [ ] 시도할 횟수를 알고 있다.
- [ ] 0과 9 사이의 무작위 값을 알고 있다.
- [ ] 주어진 값으로 경주를 할 수 있다.



### RandomNumberCreator

- [ ] 0과 9 사이의 무작위 값을 만들 수 있다.


### Racing

- [ ] 각 Car 객체가 부여받은 랜덤값을 알고 있다.
- [ ] 시도 횟수를 알고 있다.
- [ ] 각 회 마다의 경주 결과를 알 수 있다
